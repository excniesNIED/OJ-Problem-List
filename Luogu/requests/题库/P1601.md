A+B Problem（高精）
题目描述
高精度加法，相当于 a+b problem，**不用考虑负数**。
输入输出格式
输入格式

分两行输入。$a,b \leq 10^{500}$。

输出格式

输出只有一行，代表 $a+b$ 的值。
输入输出样例
输入样例 #1
1
1
输出样例 #1
2
输入样例 #2
1001
9099
输出样例 #2
10100
说明
$20\%$ 的测试数据，$0\le a,b \le10^9$；

$40\%$ 的测试数据，$0\le a,b \le10^{18}$。
