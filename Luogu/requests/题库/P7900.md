[COCI2006-2007#2] SJECIŠTA
题目描述
考虑一个有 $n$ 个顶点的凸多边形，且这个多边形没有任何三个（或以上）的对角线交于一点。

现在请你求出这样的多边形的对角线交点个数。
输入输出格式
输入格式

输入一行一个整数表示 $n$。
输出格式

输出一行一个整数表示答案。
输入输出样例
输入样例 #1
3

输出样例 #1
0
输入样例 #2
4

输出样例 #2
1
输入样例 #3
6

输出样例 #3
15
