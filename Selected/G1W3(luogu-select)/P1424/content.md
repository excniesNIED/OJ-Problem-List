Display ID

```
P1424
```

Title

```
小鱼的航程（改进版）
```

Description

```
有一只小鱼，它平日每天游泳 250 公里，周末休息（实行双休日)，假设从周 x 开始算起，过了 n 天以后，小鱼一共累计游泳了多少公里呢？
```

Input Description

```
输入两个正整数 x,n，表示从周 x 算起，经过 n 天。
```

Output Description

```
输出一个整数，表示小鱼累计游泳了多少公里。
```

Input Samples 1

```
3 10
```

Output Samples 1

```
2000
```

Hint

```
数据保证，$1≤x≤7, 1≤n≤10^6$
```
